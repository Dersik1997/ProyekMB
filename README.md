# Si Tangkulak - Ekosistem Pertanian Digital

**Si Tangkulak** adalah platform AgriTech inovatif yang dirancang untuk memperpendek rantai distribusi antara petani dan konsumen. Aplikasi ini memberdayakan petani untuk memiliki toko digital sendiri (etalase) dan memudahkan konsumen mendapatkan hasil panen segar langsung dari sumbernya.

## 🚀 Fitur Unggulan

### Bagi Konsumen
* **Katalog Panen Terpadu:** Temukan komoditas segar dengan harga yang transparan.
* **Profil & Etalase Petani:** Kenali petani di balik hasil panen melalui halaman toko khusus yang menampilkan profil dan bio.
* **Hubungan Langsung:** Transaksi aman dan cepat melalui integrasi WhatsApp langsung ke petani.
* **Informasi Kualitas:** Penanda produk untuk memudahkan pemilihan kualitas (Organik, Super, Segar).

### Bagi Petani
* **Dashboard Manajemen:** Panel kontrol mudah untuk mengelola hasil panen (tambah, edit, hapus).
* **Manajemen Identitas:** Kelola profil toko, foto, lokasi, dan deskripsi kebun.
* **Pantauan Stok:** Update volume stok secara *real-time* untuk menjaga kepercayaan pembeli.
* **Galeri Visual:** Mendukung unggahan foto produk untuk menarik minat konsumen.

## 🛠️ Stack Teknologi
* **Backend:** PHP Native
* **Frontend:** Tailwind CSS (Modern & Responsive)
* **Database:** MySQL
* **Icon Library:** FontAwesome
